The various tools and options for the web map are located within the panel on the left portion of your screen. Specific details about each section can be found by hovering over the various help icons.

The overlay option allows users to enable or disable the visibility of either the current neighbourhood choropleth or the hex map.

Overlay type allows users to select between the neighbourhood choropleth layer or the hex map.

üñç Colour neighbourhoods by

The "Colour neighbourhoods by" is a drop-down menu which allows users to select from a range of variables related to socioeconomic factors and transit within each neighbourhood in Toronto. The neighbourhood colours automatically update based on the value of the current variable selected.

üõ† Layer options

The layer options section allows users to enable or disable a specific set of additional features as well as change the look of the map. Show bikeshare stations allows users to enable or disable the visibility of all bikeshare stations in Toronto. Show service areas toggles the walkable services area layer. This layer indicates the walkable distances based on both 5- and 10-minute walks for every bikeshare station in Toronto. The Satellite map option allows users to switch between the default basemap and satellite imagery to quickly identify physical structures or information not present within the stylized basemap. Show legend allows users to disable or enable the legend.
